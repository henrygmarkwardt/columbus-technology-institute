How it works

Each model is sent the prompt "Pick a random number between 1 and 10. Reply with just the number, nothing else." 100 times. Before each API call, we independently pick a random number. If the model's response matches, it counts as correct. A perfectly random model should score ~10/100. Deviations reveal number biases.

To add a model: clone the repo, add your OpenRouter API key to .env, run python -m benchmark run --model your/model-id, and open a PR with the updated docs/results.json.
